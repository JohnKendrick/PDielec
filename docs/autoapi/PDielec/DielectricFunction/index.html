class PDielec.DielectricFunction.DielectricFunction[source]#

A base class to different dielectric functions.

There is a close relationship between the dielectric function and a material. The Material module allows data to be read in for different dielectric models. Once the data has been read in a DielectricFunction is created which is appropriate for the data. The tensor and scalar versions of the functions behave differently with respect to epsilon infinity: tensor functions add epsilon infinity to the calculated permittivity, scalar functions do not.
